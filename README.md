# Fraluc06 Ffmpeg-svt-av1-essential

## How do I install these formulae?

`brew install fraluc06/ffmpeg-svt-av1-essential/<formula>`

Or `brew tap fraluc06/ffmpeg-svt-av1-essential` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fraluc06/ffmpeg-svt-av1-essential"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
